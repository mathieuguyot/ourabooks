# ourabooks

Example project built with Angular 17 to learn... angular

Its uses:

-   Angular 17 standalone components
-   NgRx for some state management
-   supabase for user auth & data management
-   google books api to fetch some fresh book data !
-   Angular 17 SSR basics features

To try it:

-   npm install
-   install docker & then follow https://supabase.com/docs/guides/cli/local-development to create a sample supabase project
-   update key and address of the project inside environment.ts (yikes... local personal key is inside, but who cares)
-   npm run dev
